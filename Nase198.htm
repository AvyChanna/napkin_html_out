30.2  Dense sets and extension

Prototypical example for this section: Functions from ℚ → N extend to ℝ → N if they’re uniformly continuous and N is complete. See also counterexamples below.

Definition 30.2.1. Let S be a subset (or subspace) of a topological space X. Then we say that S is dense if every open subset of X contains a point of S.

Example 30.2.2 (Dense sets)

(a)
ℚ is dense in ℝ.
(b)
In general, any metric space M is dense in its completion M.

Dense sets lend themselves to having functions completed. The idea is that if I have a continuous function f : ℚ → N, for some metric space N, then there should be at most one way to extend it to a function f: ℝ → N. For we can approximate each rational number by real numbers: if I know f(1), f(1.4), f(1.41), … f() had better be the limit of this sequence. So it is certainly unique.

However, there are two ways this could go wrong:

Example 30.2.3 (Non-existence of extension)

(a)
It could be that N is not complete, so the limit may not even exist in N. For example if N = ℚ, then certainly there is no way to extend even the identify function f : ℚ → N to a function f: ℝ → N.
