52.2  Field extensions, algebraic closures, and splitting fields

Prototypical example for this section: ℚ()∕ℚ is an extension, ℂ is an algebraic closure of any number field.

First, we define a notion of one field sitting inside another, in order to generalize the notion of a number field.

Definition 52.2.1. Let K and F be fields. If F ⊆ K, we write K∕F and say K is a field extension of F.

Thus K is automatically an F-vector space (just like ℚ() is automatically a ℚ-vector space). The degree is the dimension of this space, denoted [K : F]. If [K : F] is finite, we say K∕F is a finite (field) extension.

That’s really all. There’s nothing tricky at all.

Question 52.2.2. What do you call a finite extension of ℚ?

Degrees of finite extensions are multiplicative.

Next, given a field (like ℚ()) we want something to embed it into (in our case ℂ). So we just want a field that contains all the roots of all the polynomials:

Theorem 52.2.4 (Algebraic closures)
Let F be a field. Then there exists a field extension F containing F, called an algebraic closure, such that all polynomials in F[x] factor completely.

Example 52.2.5 (ℂ)
ℂ is an algebraic closure of ℚ, ℝ and even itself.

Abuse of Notation 52.2.6. Some authors also require the algebraic closure to be minimal by inclusion: for example, given ℚ they would want only ℚ (the algebraic numbers). It’s a theorem that such a minimal algebraic closure is unique, and so these authors will refer to the algebraic closure of K.

I like ℂ, so I’ll use the looser definition.
