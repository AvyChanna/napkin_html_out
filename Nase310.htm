The key insight now is that you might think of this as a lattice: geometrically, we want to think about this the same way we think about ℤ2.

Perversely, we might try to embed this into ℚ2 by sending a + b to (a,b). But this is a little stupid, since we’re rudely making K, which somehow lives inside ℝ and is “one-dimensional” in that sense, into a two-dimensional space. It also depends on a choice of basis, which we don’t like. A better way is to think about the fact that there are two embeddings σ1 : K → ℂ and σ2 : K → ℂ, namely the identity, and conjugation:

σ1(a + b) = a + b
σ2(a + b) = a − b.
