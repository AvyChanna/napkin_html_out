Obviously, every 𝔪 can be written as 𝔪 = 𝔪0𝔪∞ by separating the finite from the (real) infinite primes.

We say a ≡ b (mod 𝔪) if

  • If 𝔭 is a finite prime, then a ≡ b (mod 𝔭ν(𝔭)) means exactly what you think it should mean: a − b ∈𝔭ν(𝔭).
  • If 𝔭 is a real infinite prime σ : K → ℝ, then a ≡ b (mod 𝔭) means that σ(a∕b) > 0.
