Remark 25.1.3 — The fact that this operation is called the “inverse” Fourier transform is mostly a historical accident (as my understanding goes). Confusingly, the corresponding quantum operation is the (not-inverted) Fourier transform.

If we apply the definition as written, computing the transform takes O(N2) time. It turns out that by an algorithm called the fast Fourier transform (whose details we won’t discuss), one can reduce this to O(N log N) time. However, for Shor’s algorithm this is also insufficient; we need something like O instead. This is where the quantum Fourier transform comes in.
