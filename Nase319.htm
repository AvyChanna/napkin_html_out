Example 51.1.2 (Examples of units in a number field)

1.
±1 are certainly units, present in any number field.
2.
If 𝒪K contains a root of unity ω (i.e. ωn = 1), then ω is a unit. (In fact, ±1 are special cases of this.)
3.
Of course, not all units of 𝒪K are roots of unity. For example, if 𝒪K = ℤ[] (from K = ℚ()) then the number 2 + is a unit, as its norm is

4.
Given any unit u ∈𝒪K×, all its powers are also units. So for example, (3 − 2)n is always a unit of ℤ[], for any n. If u is not a root of unity, then this generates infinitely many new units in 𝒪K×.
