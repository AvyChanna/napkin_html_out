The integral ∫ cα should add up all the α along the curve c.

Our differential form α first takes in a point p to get αp ∈ V ∨. Then, it eats a tangent vector v ∈ V to the curve c to finally give a real number αp(v) ∈ ℝ. We would like to “add all these numbers up”, using only the notion of an integral over [a,b].
